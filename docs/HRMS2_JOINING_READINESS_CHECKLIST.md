# HRMS2 Joining Readiness Checklist

Employee code generation is blocked until:

1. Candidate onboarding profile is HR-approved.
2. Required documents are verified.
3. BGV/eKYC is verified with no blockers.
4. Payroll HR details are validated.
5. Salary proposal is approved where applicable.
6. Salary register is locked.
7. BM / Branch Head JCLR Approval is approved.
8. Payroll HR JCLR Entry is ready or completed.
9. EPF/statutory declaration is verified.
10. Required DPDP consent is granted.
11. Existing approved-offer conversion path can return an employee code.

Readiness snapshots are stored in `joining_control_room_snapshot`.

Payroll HR cannot self-approve JCLR. Employee code remains blocked until BM / Branch Head JCLR Approval and Payroll HR JCLR Entry are both complete.
